# Context Menu
The context menu provides context-specific actions for items.
Set the <TypeLink type="CompanyOwnership" member="contextMenuItems"/> to populate the context menu with actions.
The <TypeLink type="CompanyOwnership" /> component introduces a default context menu which provides actions to interact with the chart.

```tsx
function CompanyOwnershipChart() {
  return (
    <CompanyOwnership data={data} contextMenuItems={DefaultContextMenuItems}></CompanyOwnership>
  )
}
```

To adjust the context menu visualization, add a custom React component using the <TypeLink type="CompanyOwnership" member="renderContextMenu"/> prop.
